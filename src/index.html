// Turtale.

As part of this project, my mission was to create an individual website presenting data of my choice. This data had to be stored in a MYSQL database and easily editable from the website. To do this, a "back office" had to be set up to allow an administrator to easily manage this data by adding, modifying, or deleting records. The selected data had to be complex enough to require the creation of two distinct tables in the database. For each record, a field with a photo (of the product, the movie, the singer, etc.) had to be included. Additionally, at least 50 records had to be present in one of the tables and at least 10 in the second. For one of the tables, there had to be at least 6 fields (excluding the ID) and at least 3 fields in the other (still not counting the ID).

// PLANET TOMORROW

In my portfolio, I have included a project that involved creating a blog about the environment and new technology. The task was not only to spread awareness about environmental issues but also to ensure that the website itself was the most eco-friendly possible. To achieve this, I adopted several eco-friendly practices such as using renewable energy to power the website, reducing data storage, and optimizing the website design for energy efficiency. By adopting these measures, I was able to minimize the website's carbon footprint and set an example for others to follow. This project allowed me to demonstrate my commitment to environmental responsibility while also showcasing my skills in web development and design.
